实现 fibonacci 数列，要求(以Java为例)如下：
1.不要提交依赖包等文件
2.分两次提交，第一次提交README.md，第二次提交代码
3.函数名为 Fibonacci.of()
4.of是一个静态方法，入参是int，返回值是int
5.参数从1开始，各参考值如下：
Fibonacci.of(1) == 1
Fibonacci.of(2) == 1
Fibonacci.of(3) == 2
Fibonacci.of(4) == 3
Fibonacci.of(5) == 5
...
6.在main函数用循环打印1~200的Fibonacci数列
7.提交到自己的github
8.将作业的github地址写在答案中
